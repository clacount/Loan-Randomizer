Fair Assignment Logic
  • Loan assignments are balanced across any number of loan officers.
  • Only active loan types are included in new assignments.
  • Each loan type is balanced separately to keep distribution more even.
  • The assignment logic compares loan type mix, total goal dollars, and loan count before selecting the best-fit officer.
  • Prior assignment history is included, so fairness is maintained over time, not just within a single run.
